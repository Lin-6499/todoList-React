# todoList-React
使用React+js+css实现拆分组件实现动态初始化列表

#任务功能
- ✅ 添加新任务（支持回车提交）
- ✅ 标记任务为完成/未完成
- ✅ 删除任务，删除以标记任务
- ✅ 实时统计：总任务数、已完成数、未完成数
- ✅ 组件化设计：`Header`、`List`、`Item`、`Footer`
- ✅ 使用 CSS Modules 避免样式冲突
- ✅ 数据状态由 `useState` 管理

#技术栈

| 技术 | 说明 |
|------|------|
| React | 前端框架 |
| useState | 状态管理 Hook |
| CSS Modules | 局部样式隔离 |
| JavaScript (ES6+) | 编程语言 |
